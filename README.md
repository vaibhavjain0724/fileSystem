# Mini Distributed Storage Server

Trying to build a simple distributed storage server from scratch in C++.

The goal of this project is to learn:

* C++
* Operating Systems
* Computer Networks
* Multithreading
* Linux system programming
* AWS deployment (later)

## Planned Features

* Upload files
* Download files
* Delete files
* List stored files
* Multiple client support
* Authentication
* Logging

More features will be added as I learn.
